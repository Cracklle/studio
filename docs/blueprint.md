# **App Name**: Malawi NGO Connect

## Core Features:

- Navbar: Displays the logo, navigation links (Home, About, Packages, Contact), and a 'Register NGO' button.
- Footer: Includes copyright information, social links, and quick links, consistently displayed across all pages.
- Hero Section: Presents the main value proposition with a headline, sub-headline, and clear CTAs (Get Verified, Learn More).
- Vision Section: Highlights the key benefits of the platform (Visibility, Funding, Networking) using a visual grid layout.
- Packages Preview: Introduces the available NGO packages (Basic, Standard, Premium) with a brief overview.

## Style Guidelines:

- Primary color: Deep royal blue (#1E3A8A) to reflect professionalism and trust.
- Background color: Off-white (#F9FAFB) for a clean and modern aesthetic.
- Accent color: Slate gray (#64748B) for secondary text and subtle highlights.
- Body and headline font: 'Inter', sans-serif, for a modern, machined, objective, neutral look, readable in headlines or body text. Note: currently only Google Fonts are supported.
- Clean and spacious layout to improve readability and overall user experience.
- Use of a grid system for consistent alignment and visual appeal.
- Subtle animations to improve user experience. These should be used carefully to add a visual element, not to distract.
